# 星轨清单 Orbit Todo 任务书

## 1. 项目定位

星轨清单是一款面向 Ubuntu 本地运行的创意型计划管理工具。它不是传统列表式 To Do，而是把任务、日期、灵感、倒计时和桌面便签组织成一个轻量工作台，强调活跃、青春、可玩和高可定制性。

## 2. 目标用户

- 需要把临时想法、学习计划、DDL、长期目标放在同一个空间里的学生、创作者和个人开发者。
- 喜欢视觉化、便签化、可拖拽界面的用户。
- 希望项目可一键运行、可发到 GitHub、无需后端也能长期使用的个人用户。

## 3. 核心体验

- 打开后直接进入工作台，不做营销型首页。
- 任务可以进入不同分组，并通过颜色区分。
- 输入计划时自动识别日期、时间和 DDL 关键词，并在任务卡片中高亮。
- 月历和年历都可以点击某一天，查看当天计划。
- 任务被拆成待分配、已安排、DDL、长期目标、灵感胶囊等不同状态。
- 便签浮层只显示当前计划，可拖动、缩放，并在靠近屏幕边缘时自动吸附。
- 内置番茄钟和自定义计时器。
- 提供多种 UI 风格与贴图感背景，支持即时切换。

## 4. 功能范围

### 已实现范围

- Vite + React + TypeScript 前端工程。
- 本地 `localStorage` 数据持久化，包含版本号、自动备份、跨标签页同步、导出/导入 JSON。
- 分组创建、颜色自定义、任务分组归属。
- 任务创建、完成、删除、类型切换和日期归档。
- 中文时间信息提取：今天、明天、后天、下周几、YYYY-MM-DD、MM/DD、M月D日、HH:MM、几点等。
- 时间片段高亮和任务时间标签。
- 月历视图、年历视图、日期点击筛选。
- 灵感记录和长期目标。
- 每日随机激励语。
- 可拖拽、可缩放、边缘磁吸的桌面便签浮层。
- 番茄钟和自定义计时器。
- 多主题切换。
- Electron 桌面窗口运行。
- `run.sh` 一键启动桌面应用。
- `install_desktop_launcher.sh` 安装 Ubuntu 桌面图标和应用菜单入口。

### 后续可扩展

- 系统托盘、通知、全局快捷键和系统级置顶悬浮窗。
- IndexedDB 或文件存储，用于更大规模数据。
- GitHub Pages 自动发布流程。
- 导入/导出 JSON、Markdown 周报、日历 ICS。
- 云同步、快捷键、系统通知。

## 5. 技术方案

- 前端：React + TypeScript。
- 构建工具：Vite。
- 桌面外壳：Electron。
- 图标：lucide-react。
- 数据：localStorage，带版本化 JSON 包装结构；覆盖前保留上一份自动备份；支持导出 JSON 文件到 Ubuntu 本地磁盘。
- 运行：Ubuntu 下执行 `./run.sh` 打开桌面窗口；执行 `./run_web.sh` 进入浏览器开发模式。

## 6. 数据模型

- `Task`：标题、备注、类型、状态、分组、安排日期、DDL 日期、提取时间片段、创建/更新时间。
- `Group`：名称、颜色、贴图风格。
- `Preferences`：主题、选中日期、日历视图、便签位置尺寸、计时器设置。

## 7. 验收标准

- 新机器上安装 Node.js 18+ 后，执行 `chmod +x run.sh && ./run.sh` 可启动。
- 刷新页面后任务、分组、主题、便签位置仍然存在。
- 输入包含日期的中文任务能自动显示时间标签。
- 月历和年历点击日期后能展示对应计划。
- 便签可拖动、缩放并磁吸屏幕边缘。
- `npm run build` 能通过。
